BEST BOY TANK

A top down 'Tanks' inspired game where the goal is to destroy other users tanks in a MMOG setting.


The game will mainly be developed in Javascript, but also use technologies such as Socket.IO to maintain server connections
and MySQL to manage database storage.

The development team currently consists of Callum Taylor, Jacob Mowat,
Iskander Mufazalov, Michael Soutar and Haydn McGrath.
